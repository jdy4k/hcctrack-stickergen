HCC Sticker Sheet Generator

Upload a CSV with columns name,group,sex,event,heat and this site will give you a sticker sheet PDF for Avery 5160 labels. (sample csv)

The file should end with .csv. (not .xlsx, .xls, or anything else!) Learn how to export from Excel as a CSV file here. Learn how to export from Google Sheets as a CSV file here.
